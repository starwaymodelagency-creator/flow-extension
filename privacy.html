Privacy Policy

Flow Redirect Mode is an extension that allows users to enable a redirect mode in their browser for educational and user-controlled purposes.

1. Information Collection

This extension does not collect or transmit any personal data.

2. URL Usage

URLs visited in this extension are only used locally within the browser to trigger user-defined redirects. They are not sent to any server or third party.

3. User Control

All functionality is activated only after the user explicitly enables it through the extension’s interface. The user can disable the redirect mode at any time.

4. Security

The extension does not modify or inject code into any websites. All actions are performed locally in the user’s browser.

5. Updates

We may update the extension to improve functionality. Any changes to this privacy policy will be reflected on this page.

6. Contact

If you have questions about this privacy policy, please contact us at starwaymodelagency@gmail.com.
